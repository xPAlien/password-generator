# Secure Password Generator

A modern, sleek password generator application with a dark mode UI that allows users to create strong, random passwords with customizable options.

![Password Generator Screenshot](https://imgur.com/m7MP7Em)

## ðŸ“‹ Features

- Generate 4 different passwords simultaneously
- Customize password length (4-32 characters)
- Include/exclude character types:
  - Lowercase letters (a-z)
  - Uppercase letters (A-Z)
  - Numbers (0-9)
  - Symbols (!@#$%^&*()_+-=[]{}|;:,.<>?)
- Add your own custom characters
- One-click copy to clipboard
- Visual feedback when password is copied
- Modern dark mode UI
- Mobile responsive design

## ðŸ› ï¸ Installation & Setup

### Option 1: Use the standalone HTML file

1. Clone the repository:
   ```bash
   git clone https://github.com/xPAlien/password-generator.git
   cd password-generator
   ```

2. Open `index.html` in your web browser

### Option 2: Use with React

If you want to use the React version:

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

### GitHub Pages

This project is configured for easy deployment to GitHub Pages:
